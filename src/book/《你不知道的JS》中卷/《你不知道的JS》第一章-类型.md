《你不知道的JS》中卷 第一部分 类型和语法

# 第一章 类型

## 1. 内置类型

- boolean
- number
- string
- null
- undefined
- object
- symbol

函数，数组都是对象的子类型。


## 2. 值和类型

typeof运算符总是会返回一个字符串。

已在作用域中声明但是还没有赋值的变量，是undefined。还没有在作用域中声明的变量是undeclared。

## 3. 小结

JS有7种内置类型：string, number, boolean, undefined, null, object, symbol, 可以使用typeof运算符来查看。undefined是值的一种，undeclared表示变量还没有被声明过。
